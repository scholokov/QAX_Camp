Описание курса:

Практический курс, который включает в себя основы программирования на Java, а также разбор основных вопросов по написанию UI-автотестов для Web-приложений. Курс подойдёт для тех, кто хочет заниматься автоматизацией тестирования, но пока не знает с чего начать.

Минимальные требования:

● Базовые знания по тестированию.
● Уверенный пользователь ПК.
● Наличие собственного ноутбука или ПК для занятий.
● Установленное ПО: Intellij IDEA, Java 8+, Maven.

После курса вы сможете:

● Писать UI-автотесты с помощью языка программирования Java и стэка самых современных и востребованных технологий для автоматизации тестирования (Selenide/Selenium + Maven/Gradle + jUnit/TestNG).
● Создавать собственный фреймворк с нуля.
●Настраивать VM в Google Cloud, поднимать на ней Jenkins и настраивать CI/CD систему с Docker и Selenoid.
● Генерировать чёткие, понятные и презентабельные отчеты с помощью Allure Framework.
● Автоматизировать рутинные задачи, имитируя основные действия пользователя в браузере.
● Претендовать на позицию Junior QA Automation Engineer.
Урок Тема Опис
1 Знакомство и общая теория по Java Что такое класс, метод, переменная. Зачем нужны модификаторы доступа. Написание первой программы “Hello world!”
2 Java Циклы. Условные операторы. Статика. Преобразование типов.
3 Java Массивы и коллекции
4 Java ООП: полиморфизм, наследование, инкапсуляция, абстракция
5 Знакомство с jUnit и TestNG Основные аннотации. Написание юнит тестов с использованием библиотеки Assert
6 Знакомство с Maven и Gradle Создание проектов и подключение основных фреймворков и библиотек
7 Знакомство с Git и GitHub Генерация ключей. Команды clone, pull, commit, push и др.
8 Error handlers. Loggers. Listeners Обработка исключений. Создание и настройка собственного логера и листенара
9 Selenide и Allure Создание структуры фреймворка,настройка базовых библиотек и написание тестов. Разбор чётких и понятых отчётов
10 X-Path и CSS. Parameter Injections Практика написания селекторов. Параметризация тестов через TestNG.xml файл
11 Паттерн проектирования автотестов "PageObject" Написание теста с использованием паттерна PageObject
12 Явные и неявные ожидания. Retry analyzer Создание и настройка собственного ретрайера для упавших тестов
13 Работа с Excel и CSV файлами. Data Provider Написание методов для работы с Excel и CSV файлами. Использование в тестах фабрику тестовых данных Data Provider
14 Знакомство с Docker, Selenoid Подключение докер технологий к автотестам. Запуск браузера в контейнере
15 Облачные технологии в автоматизации тестирования Настройка VM и поднятие Jenkins в Google cloud
16 Облачные технологии в автоматизации тестирования Настройка Continious Integration & Shedulers
